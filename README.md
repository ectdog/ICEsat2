# ICEsat2
combine ICEsat2 and Sentinel-2 extract elevation
Code: Representative modules for the three processing stages — ATL03 photon preprocessing (atl03_config.py, atl03_pipeline.py), model training (config.py, data_pipeline.py, model.py, trainer.py), and DEM inference (infer_config.py, infer_pipeline.py) — are provided to support reproducibility. Complete pipeline execution requires the full multi-year dataset and trained model checkpoint, which are available from the corresponding author upon reasonable request.
